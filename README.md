# OASIS INFOBYTE

This folder contains a collection of simple web development projects created for practice and demonstration purposes.

## Projects Included

### 1. Landing Page
A basic landing page design with a clean layout and styling. It is a great example of a simple promotional website.

### 2. Portfolio
A personal portfolio website that showcases profile content and projects in a structured web page.

### 3. Temperature Converter Website
A small interactive web app that converts temperatures between Celsius, Fahrenheit, and Kelvin.

## Folder Structure

- Landing Page/
  - index.html
  - style.css
  - launch_project.bat
- Portfolio/
  - index.html
  - style.css
  - app.js
  - assets/
- Temperature converter website/
  - index.html
  - style.css
  - script.js

## How to View the Projects

Each project is a standalone website. To view one:

1. Open the relevant folder.
2. Open the index.html file in any web browser.
3. If available, you can also run the batch file in the Landing Page folder to launch it.

## Technologies Used

These projects are built using:

- HTML
- CSS
- JavaScript

## Purpose

This folder is intended for learning front-end web development, practicing layout design, and building small interactive web applications.
